# Wajiha Theme Tools

Monorepo for the Wajiha theme development toolkit.

## Packages

- **[@wajiha/theme-sdk](packages/theme-sdk/)** — React hooks, components, and types for building themes
- **[@wajiha/cli](packages/cli/)** — CLI tool for theme development (`wajiha init`, `wajiha build`, `wajiha push`)

## Examples

- **[default-theme](examples/default-theme/)** — Reference theme implementation (Midnight Prism)

## Getting Started

```bash
npm install
npm run build
```
